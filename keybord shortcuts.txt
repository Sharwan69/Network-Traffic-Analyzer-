Here are some common keyboard shortcuts for Windows laptops: 
Ctrl + C: Copy selected text
Ctrl + X: Cut selected text
Ctrl + V: Paste copied or cut text
Ctrl + Z: Undo the last action
Ctrl + Y: Redo the last action
Ctrl + A: Select all text in the active window or on the page
Ctrl + B: Bold the selected text
Ctrl + I: Italicize the selected text
Ctrl + U: Underline the selected text
Ctrl + F: Find text in a document or file
Ctrl + T: Open a new tab in a web browser
Ctrl + W: Close the current window
Ctrl + S: Save the current document
Ctrl + P: Print the current document
Ctrl + N: Create a new document or file
Ctrl + O: Open a document or file
Alt + Tab: Switch between open applications
Alt + F4: Close the current program or window
Windows key + D: Show or hide the desktop
F5: Refresh the active window
F6: Cycle through elements in a window or on the desktop
F10: Activate the menu bar in the active window
